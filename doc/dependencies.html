Dependencies

To be able to run DIVE you have to install boogie.
For Ubuntu since version 18 this can be done with:

sudo apt install boogie

For other systems please refer to the Boogie Github

Java Version

DIVE runs with the Oracle JRE 8 only (not openJDK). To download the JRE click here.

Run DIVE

To run dive simply run the provided jar. For unix systems:

java -jar DIVE.jar

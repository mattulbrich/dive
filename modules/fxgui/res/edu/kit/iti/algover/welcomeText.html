How to start?

There are three ways to start:

  1. To open an existing DIVE project using a project configuration press the button Open project.
  2. To create a new DIVE project using existing Dafny programs press the button New project.
  3. To create an empty DIVE project with an empty Dafny program press the button Empty project.
  4. To open the linked list example of the paper at VSTTE 2019 press the button Load Example.
When loading a project with existing Dafny files, DIVE will generate the verifications conditions upon startup.

If the Dafny code is changed, the verification conditions need to be regenerated using the button Refresh.

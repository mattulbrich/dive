How to start?

There are three ways to start:

  1. Create an empty project with an empty Dafny program, select the button Empty project
  2. Create a new project using existing Dafny programs, select the button Create project
  3. Open an existing project using an project configuration, select the button Open project
When loading a project with existing Dafny files, AlgoVer will generate the verifications conditions upon startup.

If the Dafny code is changed, the verification conditions need to be regenerated using the button Refresh.
